Detailed Description

Simple utility class that provides basic arithmetic operations for two System.Double valuess.

This class exposes a single static operation method intended for small console examples and documentation. It is intentionally minimal: callers provide two operands and an operation code.

double sum = Calculator.DoOperation(5.0, 3.0, "a"); // returns 8.0
double quotient = Calculator.DoOperation(10.0, 0.0, "d"); // returns double.NaN (division by zero)

Provides simple arithmetic operations for two System.Double values.

Supported operations: 'a' - Add, 's' - Subtract, 'm' - Multiply, 'd' - Divide (returns System.Double if divisor is zero). The method returns System.Double for unrecognized operation codes.
